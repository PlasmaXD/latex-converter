以下は微分方程式の代表的な例です。

1. **一般形の1階常微分方程式**  
   
$$$$ 
   \frac{dy}{dx} = f(x,y)
    $$$$


2. **1階線形微分方程式**  
   
$$$$ 
   \frac{dy}{dx} + p(x)y = q(x)
    $$$$


3. **2階線形常微分方程式（斉次形）**  
   
$$$$ 
   \frac{d^2 y}{dx^2} + p(x)\frac{dy}{dx} + q(x)y = 0
    $$$$


4. **2階線形常微分方程式（非斉次形）**  
   
$$$$ 
   \frac{d^2 y}{dx^2} + p(x)\frac{dy}{dx} + q(x)y = g(x)
    $$$$


これらは基本的な形ですが、問題の種類に応じてさまざまな形が考えられます。
